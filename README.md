# Breast-Cancer-Classifiaction-using-BUSI-Dataset-Kaggle-
Breast cancer multi-class classification using EfficientNet-B4 with attention. After strict no-overlap splitting, training data was balanced via oversampling. Advanced training (Mixup, AdamW, AMP, SWA, early stopping) achieved 98.31% test accuracy with strong generalization.
